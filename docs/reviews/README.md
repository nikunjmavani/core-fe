# Review artifacts

Generated audit outputs land here (not committed by default unless the run explicitly saves them).

| Command                             | Output                                            |
| ----------------------------------- | ------------------------------------------------- |
| `pnpm report:code-review`           | Full-repo review report (markdown)                |
| `/pre-merge-review` (agent command) | Ad-hoc PR review notes — save here when requested |

Keep filenames dated: `YYYY-MM-DD-<topic>.md`.
