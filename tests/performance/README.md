# Performance Tests

Lighthouse CI, bundle-size assertions, or front-end k6 scenarios.

**Add here when:** You add performance gates (e.g. Lighthouse budget, `bundlesize` or similar, or k6 scripts that hit the front-end). General load/smoke stays in `tests/load/`.

**Tools:** Depends on what you add (Lighthouse CI, bundlesize, k6, etc.). Document the command in this README and in `tests/README.md` when you add specs.
